npm-app-project-template
========================

Project template for a NPM project.   

A very basic project that illustrates how to create a npm project.  Project is tested using a BDD approach.

#### Included Libraries

* jquery 2.1.1
* underscore 1.6.0

#### Grunt Commands

* <code>grunt lint</code> runs the JavaScript file linter.
* <code>grunt test</code> runs the unit test
* <code>grunt</code> runs a local server on http://localhost:8000
* <code>grunt production</code> creates distribution files of javascript and css

#### Publishing NPM Project

Create an account on https://www.npmjs.org/ and then at the root directory of your project type <code>npm publish .</code> to publish project.
